# 📋 Cadastro de Pessoas em Python

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de criar um sistema simples de cadastro de pessoas utilizando arquivos de texto. Ele permite armazenar e visualizar informações de usuários de forma prática diretamente no terminal.

## 📄 Descrição

O sistema funciona por meio de um menu interativo, onde o usuário pode escolher entre as seguintes opções:

* **1 - Cadastrar pessoa:**
  Permite inserir nome, telefone e e-mail, salvando essas informações em um arquivo chamado `pessoas.txt`.

* **2 - Listar pessoas:**
  Exibe todos os registros salvos no arquivo, mostrando os dados de forma organizada.

* **3 - Sair:**
  Encerra o programa.

Os dados são armazenados no formato:

```
nome,telefone,email
```

O projeto utiliza conceitos importantes da programação em Python, como:

* Estrutura de repetição (`while`)
* Estrura condicional (`if`, `elif`)
* Manipulação de arquivos (`open`, `write`, `read`)
* Entrada de dados (`input`)
* Biblioteca `os` para limpar a tela
